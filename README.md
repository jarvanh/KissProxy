# Kiss Proxy for Android

## Features
* Simple API
* Java NIO based
* Single thread
* HTTP&HTTPS support
* High performance

## Sample code
```
ProxyServer.getInstance().start();
ProxyServer.getInstance().stop();
ProxyServer.getInstance().isRunning();
ProxyServer.getInstance().getPort
```
Default port

8964
